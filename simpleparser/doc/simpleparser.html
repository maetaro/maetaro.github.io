simpleparser.none_of(s)¶

none_of function.

As the dual of oneOf, none_of(cs) succeeds if the current character not in the supplied list of characters cs. Returns the parsed character.

Example

>>> from simpleparser import none_of, choice, token, many, transform, seq
>>> p = none_of("abcdefg")
>>> p.exec("hello")
['h']
>>> chars = choice(token('""'), none_of('",'))
>>> p = transform(many(chars), lambda x: ["".join(x)])
>>> text = r'Shirt with ""Haskell"" text'
>>> p.exec(text)
['Shirt with ""Haskell"" text']
>>> dq = token('"')
>>> p2 = transform(seq(dq, p, dq), lambda x: ["".join(x)])
>>> text = r'"Shirt with ""Haskell"" text"'
>>> p2.exec(text)
['"Shirt with ""Haskell"" text"']

simpleparser.many(parser)¶

Many function.

Receives one parser object. And repeats parsing for success. Must succeed at least once.

Parameters

parser (simpleparser.parser.Parser) – The Parser object.

Returns

Generated new Parser object.

Return type

Parser

Example

>>> from simpleparser import many, token
>>> p = many(token("foo"))
>>> p.exec('foo')
['foo']
>>> p.exec('foofoo')
['foo', 'foo']
>>> p.exec('bar')
parse error at (0): unexpected bar expecting foo (by token)
>>> p.exec('foobar')
['foo']

simpleparser.transform(parser, selector)¶

Transform function.

Receives a parser object and a function that transforms the parse results. Apply the received function to the parse result and return it.

Parameters

  • parser (simpleparser.parser.Parser) – The Parser object.

  • selector (Callable[[List[str]], Any]) – The function that formats and returns the parse result.

Returns

Generated new Parser object.

Return type

Parser

Example

>>> from simpleparser import token, transform
>>> p = transform(token("foo"), lambda x: [",".join(x) + " aaa"])
>>> p.exec("foo")
['foo aaa']

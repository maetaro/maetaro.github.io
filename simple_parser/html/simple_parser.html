simple_parser.simple_parser.endBy(parser: simple_parser.simple_parser.Parser, sep: simple_parser.simple_parser.Parser) → simple_parser.simple_parser.Parser¶

The endBy p sep parses zero or more occurrences of p, separated and ended by sep.

Returns a list of values returned by p.

Example

>>> endBy(regex('\w*'), token(',')).exec('').result()
['']
>>> endBy(regex('\w*'), token(',')).exec('hoge,hoge').result()
['hoge', 'hoge']
>>> endBy(regex('\w*'), token(',')).exec('hoge,hoge,').result()
['hoge', 'hoge', '']
>>> endBy(regex('\w*'), token(',')).exec('hoge,hoge,-').result()
'parse error.'

simple_parser.simple_parser.noneOf(s: str) → simple_parser.simple_parser.Parser¶

As the dual of oneOf, noneOf cs succeeds if the current character not in the supplied list of characters cs. Returns the parsed character.

Example

>>> noneOf("abcdefg").exec("hello", 0).result()

simple_parser.simple_parser.regex(pattern: str) → simple_parser.simple_parser.Parser¶

Regex function returns a function that parses the beginning of the received string with the regular expression pattern.

Parameters

pattern (str) – a regular expression string.

Example

>>> parser = regex("hoge")
>>> parser.exec('hoge', 0).result()
['hoge']
>>> parser = regex("([1-9][0-9]*)")
>>> parser.exec('2014a', 0).result()
['2014']
>>> parser.exec('01', 0).result()
'parse error at (0): unexpected 01 expecting ([1-9][0-9]*)'
